# Assets Folder

App icons, splash screens, and brand images for the Glory to God PPEC app.

## Brand colors

- Primary: `#1A5276` (navy) — used for splash background and adaptive icon background
- Accent: `#F1C40F` (gold)

Configured in [../app.json](../app.json).

## Required images

1. **gtg_logo.png** — app icon + adaptive icon foreground (1024×1024, transparent background recommended)
2. **splash.png** — splash screen image (1242×2436). Background color `#1A5276` is set in `app.json`.
3. **favicon.png** — web favicon (48×48 or 96×96)

## Updating the logo

Replace `gtg_logo.png` with a new PNG at the same dimensions and the build will pick it up.
